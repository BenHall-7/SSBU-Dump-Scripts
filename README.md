# SSBU-Dump-Scripts
Script to dump character game animcmd scripts programatically using r2pipe (3.0.0 only)

## Requirements
* Python 3
* pip install r2pipe

## Usage
```
python main.py ".elf path"
python main.py "directory with .elf files"
```
